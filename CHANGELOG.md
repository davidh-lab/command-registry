# CHANGELOG

## 1.0.0 — 2026-07-08

### Added
- Eerste versie van het externe commandregister.
- `commands.json` toegevoegd als machineleesbare bron van waarheid.
- `command-library.md` toegevoegd als mensleesbare commanddocumentatie.
- `fallback-knowledge-instructie.md` toegevoegd voor tijdelijke Knowledge fallback.
- `commands.schema.json` toegevoegd voor basisvalidatie.
- `validation-summary.json` toegevoegd als validatieverslag.

### Included
- Algemene slash-commands uit de commandbibliotheek.
- Gecombineerde commandregels.
- Doorgangsbesluit-terminologie.
- `/consolideer`.
- Deprecated registratie van `/feedback`.

### Deprecated
- `/feedback` is niet actief wegens platformconflict.
- Gebruik `/terugkoppeling` als vervangend command.
